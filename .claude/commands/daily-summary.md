# /daily-summary

> End-of-day wrap-up and tomorrow preparation.

## Routing

Execute this skill by combining:
1. **Agent persona**: `@execution-agent`
2. **Skill instructions**: `skills/execution/daily-summary.md`
3. **User context**: `agents/_shared/context.md`

## What This Does

Creates an end-of-day summary including:
- Achievement score and completion rate
- Documented accomplishments and outcomes
- Rolled items with reasons
- Key learnings
- Tomorrow's draft priorities

## Quick Use

Just run `/daily-summary` and share:
- What got done today
- What didn't get done and why
- Any wins or challenges

The agent captures learnings and sets you up for tomorrow.
