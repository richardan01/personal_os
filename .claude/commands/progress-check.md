# /progress-check

> Mid-day progress assessment and afternoon adjustment.

## Routing

Execute this skill by combining:
1. **Agent persona**: `@execution-agent`
2. **Skill instructions**: `skills/execution/progress-check.md`
3. **User context**: `agents/_shared/context.md`

## What This Does

Performs a mid-day check including:
- Progress score against morning plan
- Completed vs. pending tasks
- Recommended adjustments
- Revised afternoon plan

## Quick Use

Just run `/progress-check` and share:
- What you've completed so far
- Any blockers or unexpected issues
- Current energy level

The agent provides honest assessment and actionable adjustments.
