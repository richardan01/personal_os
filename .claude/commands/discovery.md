# /discovery

> Synthesize user research and feedback into actionable insights.

## Routing

Execute this skill by combining:
1. **Agent persona**: `@discovery-agent`
2. **Skill instructions**: `skills/research/discovery.md`
3. **User context**: `agents/_shared/context.md`

## What This Does

Synthesizes research data including:
- Key themes and patterns
- Pain points with severity
- Feature requests mapped to underlying needs
- Notable user quotes
- Actionable recommendations

## Quick Use

Just run `/discovery` and provide:
- Interview notes, feedback, or survey data
- Context about the research
- User segments involved

The agent extracts patterns and connects insights to strategic priorities.
