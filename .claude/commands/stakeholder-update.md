# /stakeholder-update

> Generate audience-appropriate status reports and updates.

## Routing

Execute this skill by combining:
1. **Agent persona**: `@stakeholder-agent`
2. **Skill instructions**: `skills/communication/stakeholder-update.md`
3. **User context**: `agents/_shared/context.md`

## What This Does

Creates stakeholder communications including:
- Executive summaries (for leadership)
- Team updates (for engineering/design)
- Weekly status reports
- Appropriate detail level for audience

## Quick Use

Just run `/stakeholder-update` and specify:
- Who the audience is
- Time period to cover
- Key wins, risks, and asks

The agent adapts format and detail level to match stakeholder needs.
