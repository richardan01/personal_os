# /strategy-check

> Assess alignment between current work and strategic goals.

## Routing

Execute this skill by combining:
1. **Agent persona**: `@strategy-agent`
2. **Skill instructions**: `skills/planning/strategy-check.md`
3. **User context**: `agents/_shared/context.md`

## What This Does

Performs a strategy alignment check including:
- OKR progress snapshot
- Alignment score for current activities
- Identification of misaligned work
- Strategic recommendations
- Questions to consider

## Quick Use

Just run `/strategy-check` and share:
- What you're currently working on
- Recent accomplishments
- Planned work for coming week

The agent maps activities to OKRs and recommends focus areas.
