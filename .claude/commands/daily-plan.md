# /daily-plan

> Generate a comprehensive morning daily execution plan.

## Routing

Execute this skill by combining:
1. **Agent persona**: `@execution-agent`
2. **Skill instructions**: `skills/execution/daily-plan.md`
3. **User context**: `agents/_shared/context.md`

## What This Does

Creates a prioritized daily plan including:
- Top 3 priorities linked to OKRs
- Time-blocked schedule
- Meeting preparation
- Success criteria

## Quick Use

Just run `/daily-plan` and provide:
- Today's meetings and commitments
- Most important tasks
- Any blockers or deadlines

The agent handles the rest.
