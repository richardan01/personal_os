# Shared Output Formats

> Reusable templates for consistent output across all skills.

---

## Daily Execution Templates

### Daily Plan Template

```markdown
## Daily Execution Plan - [Date]

### Day Overview
- **Focus Time**: [X hours]
- **Meetings**: [Count] ([Y hours])
- **Energy Forecast**: [High/Medium/Demanding]

### Top 3 Priorities
1. **[Task]** - [Why it matters] - [Time estimate]
2. **[Task]** - [Why it matters] - [Time estimate]
3. **[Task]** - [Why it matters] - [Time estimate]

### Time-Blocked Schedule
| Time | Activity | Type |
|------|----------|------|
| 8:00-9:00 | [Task] | Focus/Meeting/Admin |

### Success Criteria
Today is successful if:
1. [Measurable outcome]
2. [Measurable outcome]
3. [Measurable outcome]
```

### Progress Check Template

```markdown
## Progress Check - [Time]

### Score: [X/10]
[Brief explanation]

### Completed
- [Task] - [Impact: High/Medium/Low]

### In Progress
- [Task] - [Status/blockers]

### Adjustments Needed
- [What to change for remaining time]

### Revised Plan
1. [Next priority] - [Time block]
2. [Next priority] - [Time block]
```

## Planning Templates

### Sprint Plan Template

```markdown
## Sprint Plan - Sprint [X]

### Sprint Goal
[One clear, measurable objective]

### Capacity
- Team Size: [X]
- Sprint Length: [X days]
- Net Capacity: [X points/hours]

### Committed Items

#### P0 - Must Complete
| Story | Points | Owner | Dependencies |
|-------|--------|-------|--------------|

#### P1 - Should Complete
| Story | Points | Owner | Dependencies |
|-------|--------|-------|--------------|

### Dependencies & Risks
| Item | Depends On | Risk | Mitigation |
|------|------------|------|------------|

### Success Criteria
1. [Measurable outcome]
2. [Measurable outcome]
```

### Strategy Check Template

```markdown
## Strategy Alignment Check - [Date]

### Alignment Score: [X/10]

### OKR Progress
| Objective | Key Result | Progress | Status |
|-----------|------------|----------|--------|

### Current Work Alignment
**Aligned:**
- [Activity] -> [Which OKR]

**Potentially Misaligned:**
- [Activity] - [Question]

### Recommendations
- **This Week**: [Focus area]
- **This Month**: [Larger initiative]
```

## Research Templates

### Discovery Insights Template

```markdown
## Discovery Insights - [Topic/Date]

### Summary
[2-3 sentence overview]

### Key Themes
1. **[Theme]**: [Description]
   - Evidence: [Quote or data]
   - Frequency: [How often]

### Pain Points
| Pain Point | Severity | Frequency | Segment |
|------------|----------|-----------|---------|

### Feature Requests (Underlying Needs)
| Request | Real Need | JTBD | Priority |
|---------|-----------|------|----------|

### Notable Quotes
> "[Quote]" - [Context]

### Recommendations
1. **Quick Win**: [Action]
2. **Investigate**: [Research needed]
3. **Strategic**: [Bigger consideration]
```

## Quality Checklist

All outputs should:
- [ ] Be actionable with clear next steps
- [ ] Include measurable success criteria where applicable
- [ ] Connect to strategic priorities/OKRs
- [ ] Be appropriately detailed for the audience
- [ ] Flag blockers and risks proactively
